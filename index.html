Installation (Ubuntu)

Download maitred-0.1.tar.gz. Then, in the terminal (in the same directory as the downloaded tar file):

sudo apt-get install python3 python3-setuptools
sudo easy_install3 maitred-0.1.tar.gz

Quick Start

Download the sample.cfg and pingpong.sh executable. In the same directory, run:

maitred sample.cfg

Check the #maitredtest channel on irc.freenode.net for the maitredtest user, which should reply to any "ping" with a simple "pong" response.
